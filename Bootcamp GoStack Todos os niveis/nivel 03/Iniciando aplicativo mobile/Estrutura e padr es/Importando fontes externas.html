Importando fontes externas

Importar fontes que nao sao padrao do sistema!

https://fonts.google.com/specimen/Roboto+Slab?sidebar.open&selection.family=Roboto+Slab:wght@400;500

Baixar a fonte

Extrair o arquivo baixado

Ir na pasta static e copiar a fonte Medium e Regular

E colar na pasta fonts, que vai ser criada.

Criar uma pasta na raiz do projeto chamada assets dentro criar a pasta fonts e colocar dentro as fontes copiadas

assets/fonts/RobotoSlab-Medium.ttf

assets/fonts/RobotoSlab-Regular.ttf

No Android usar o nome da fonte baixada no font-family

No IOS, instalar as fontes no sistema, ir no catalogo de fonte e ver o nome da fonte, PostScriptName.

Na raiz criar um arquivo chamado react-native.config.js

react-native.config.js

module.exports = {
  project: {
    ios: {},
    android: {},
  },
  assets: ['./assets/fonts/'],
};

Agora no terminal executar o comando

yarn react-native link

Colocar um texto no SignIn

SignIn/index.tsx

import React from 'react';
import { Image } from 'react-native';

import { Container, Title } from './styles';

import logoImg from '../../assets/logo.png';

const SignIn: React.FC = () => {
  return (
    <Container>
      <Image source={logoImg} />
      <Title>Faça seu logon</Title>
    </Container>
  );
};

export default SignIn;
