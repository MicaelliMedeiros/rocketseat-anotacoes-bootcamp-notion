Configurando estrutura

Desinstalar globamente o react-native-cli, se estiver instalado globalmente e instalar a nova cli

npm uninstall -g react-native-cli

npm i -g @react-native-community/cli

Iniciar o projeto

npx react-native init appgobarber --template react-native-template-typescript

Iniciar o projeto no emulador ou celular, dois terminais

yarn start

yarn android

Na raiz criar a pasta src e dentro o arquivo App.tsx

App.tsx

import React from 'react';
import {View} from 'react-native';

const App: React.FC = () => <View />;

export default App;

e no index.js da raiz, mudar a origem do App para src/App

index.js

/**
 * @format
 */

import {AppRegistry} from 'react-native';
import App from './src/App';
import {name as appName} from './app.json';

AppRegistry.registerComponent(appName, () => App);

E apagar o App.tsx da raiz.

E apagar o arquivo .eslintrc.js e .prettierrc.js
