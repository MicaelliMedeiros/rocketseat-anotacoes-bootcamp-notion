Configurando o projeto

Criar a pasta Typescript e rodar 

yarn init -y
yarn add typescript -D

Criar a pasta src e dentro o arquivo index.ts

Instalar o express

yarn add express

Para instalar as tipagens do express: Passar o mouse em cima do três pontinhos.

yarn add @types/express -D

E importar o express no index.ts

import express from "express";

const app = express();

app.get("/", (request, response) => {
  return response.json({ message: "Hello World" });
});

app.listen(3333);

Converter o arquivo TypeScript em JavaScript para o Node entender e rodar.

yarn tsc src/index.ts

Para criar o tsconfig.json

yarn tsc src/index.ts

Agora ele vai pegar automaticamente todos os arquivos ts e converter em js.

yarn tsc src/index.ts

E agora só rodar o node e acessar o localhost:3333

node src/index.js

Colocar o arquivo js convertido em uma pasta chamada dist. Ir no tsconfig.json e descomentar a linha outDir e deixar "./dist":

"outDir": "./dist" /* Redirect output structure to the directory. */,

E rodar para criar a pasta dist com o arquivo convertido dentro.

yarn tsc
