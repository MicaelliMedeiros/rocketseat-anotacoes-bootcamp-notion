Criar a tabela de appointments

No ormconfig.json configurar uma pasta onde serao armazenadas as migrations do banco de dados.

Criar na pasta database a pasta migrations

{
  "type": "postgres",
  "host": "localhost",
  "port": 5432,
  "username": "postgres",
  "password": "docker",
  "database": "gostack_gobarber",
  "migrations": ["./src/database/migrations/*.ts"],
  "cli": {
    "migrationsDir": "./src/database/migrations"
  }
}

Ir no package.json e criar um script para fazer o typeorm salvar no formato de TypeScript.

package.json

{
  "name": "primeiro-projeto-node",
  "version": "1.0.0",
  "main": "index.js",
  "license": "MIT",
  "scripts": {
    "build": "tsc",
    "dev:server": "ts-node-dev --inspect --transpileOnly --no-notify --ignore-watch node_modules src/server.ts",
    "typeorm": "ts-node-dev ./node_modules/typeorm/cli.js"
  },
}

Verificar se esta funcionando:

yarn typeorm

Para criar um migration no TypeORM

yarn typeorm migration:create -n CreateAppointments

Agora ver na pasta de migrations, a migration CreateAppointments criada.

E para executar as migrations, para serem criadas no banco de dados

yarn typeorm migration:revert // DESFAZER AS MIGRATIONS
