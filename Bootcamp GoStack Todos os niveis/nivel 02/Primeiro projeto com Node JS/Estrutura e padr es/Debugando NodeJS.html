Debugando NodeJS

Ir no Debug do VSCode e clicar em create a launch.json file

launch.json

{
  //Use IntelliSense to learn about possible attributes.
  // Hover to view descriptions of existing attributes.
  // For more information, visit: https://go.microsoft.com/fwlink/?linkid=830387
  "version": "0.2.0",
  "configurations": [
    {
      "type": "node",
      "request": "attach",
      "protocol": "inspector",
      "restart": true,
      "name": "Debug",
      "skipFiles": [
        "<node_internals>/**"
      ],
      "outFiles": [
        "${workspaceFolder}/**/*.js"
      ]
    }
  ]
}

Ir no packages.json e adicionar a flag --inspect no script do dev:server

"dev:server": "ts-node-dev --inspect --transpileOnly --ignore node_modules src/server.ts"

Para iniciar o server junto com o debug

yarn dev:server

E como funcionar o Debug?

index.ts

import { Router } from 'express';

const routes = Router();

routes.post('/users', (request, response) => {
  const { name, email } = request.body;

  const user = {
    name,
    email,
  };

  return response.json(user);
});

export default routes;

Executar o código pelo Insomnia:

Criar uma rota POST: http://localhost:3333/users

JSON:

{
	"name": "Diego",
	"email": "diego@rocketseat.com.br"
}

Dar Send, logo ele vai dar erro.

Ir ate const { name, email } = request.body; e clicar na bolinha vermelha de debug.

E debugar denovo, ele vai dar erro.

Porque ele não ta entendendo o formato JSON.

Ir ate o server.ts e adicionar o app.use(express.json());

server.ts

import express from 'express';
import routes from './routes';

const app = express();

app.use(express.json());
app.use(routes);

app.listen(3333, () => {
  console.log('🚀️ Server started on port 3333!');
});

Cancelar a requisição no Insomnia, reiniciar o servidor.

E fazer a requisição novamente no Insomnia, e o debug vai mostrar o resultado.

Para facilitar o debug, você pode escrever no WATCH do debug.

request.body, request.query, e ele vai trazer a informações que você quer.
